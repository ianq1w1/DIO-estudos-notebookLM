# DIO-estudos-notebookLM
# ⚡ Miniguia de Estudos: Eletrônica Digital e Arquitetura de Sistemas

Este repositório foi desenvolvido para o desafio de projeto da DIO, utilizando o **Google NotebookLM** como uma ferramenta de aprendizagem ativa. O foco deste trabalho é a consolidação de conhecimentos fundamentais de eletrônica, integrando desde a análise de circuitos analógicos até o design de hardware complexo com VHDL.

## 🎯 Contexto e Objetivos

O objetivo deste caderno temático é criar uma base sólida que conecte a teoria de circuitos à implementação prática em hardware programável. Como desenvolvedor focado em sistemas embarcados e IoT, compreender o fluxo que vai da análise de malhas à lógica digital é essencial para projetar dispositivos mais eficientes e otimizados.

**Objetivos de Estudo:**
* Dominar técnicas de análise de circuitos para aplicação direta em projetos de hardware.
* Consolidar os princípios de lógica digital e sistemas combinacionais/sequenciais.
* Estruturar o fluxo de desenvolvimento em linguagens de descrição de hardware (VHDL).

## 📚 Curadoria de Fontes

Para alimentar a base de conhecimento no NotebookLM, foram selecionadas as seguintes obras:

1. **Análise de Circuitos**: Base teórica para leis de Kirchhoff e comportamento de componentes passivos.
2. **Eletrónica Digital**: Foco em portas lógicas, flip-flops e a arquitetura básica de processadores.
3. **Electronics For Dummies**: Utilizado para simplificar conceitos complexos e obter uma visão prática de prototipagem e componentes.
4. **VHDL - Descrição e Síntese de Circuitos Digitais**: Referência técnica para a tradução de lógica em código para hardware programável (FPGA/CPLD).

## 📝 Miniguia de Estudo (Entrega Final)

### Resumo Estruturado
1. **Fundamentos**: A análise de circuitos fornece a base para entender a integridade de sinal em sistemas digitais e o comportamento da energia no hardware.
2. **Lógica Combinacional**: As portas lógicas são os átomos da computação; entender sua transição para sistemas sequenciais é o que permite a criação de memórias.
3. **VHDL**: Diferente de linguagens de software, o VHDL descreve o comportamento físico e o paralelismo do hardware, exigindo um pensamento voltado ao design de circuitos.

### 📖 Glossário de Conceitos
* **VHDL**: *VHSIC Hardware Description Language* – Linguagem para modelagem de sistemas digitais.
* **Lei de Ohm**: Relação fundamental entre tensão (V), corrente (I) e resistência (R).
* **Flip-Flop**: Elemento básico de armazenamento de memória em circuitos sequenciais.
* **Sinal PWM**: Técnica de modulação para controle de potência em componentes como motores e LEDs.
